# SpaceMountain Platform Manual

The documentation in `docs/` explains SpaceMountain as a creator operating system.

Start with:

1. `docs/DOCS_HOME.md`
2. `docs/VISION.md`
3. `docs/ARCHITECTURE.md`
4. `docs/creator/GETTING_STARTED.md`
5. `docs/developers/BUILD_YOUR_FIRST_APP.md`

Core idea:

SPMT is the Creator Cloud.
SpaceMountain.live is the Command Bridge.
Apps are connected modules.
Athena is the Operating Intelligence.
The Event Bus lets apps work together without hard-coded coupling.
