# Known Limitations

SpaceMountain is evolving quickly. Some surfaces may be MVP, alpha, or scaffolded.

## Current Areas To Label Clearly

- Some app integrations may still require adapters.
- Some apps may not fully restore SPMT identity yet.
- Some platform developer features may exist as MVP surfaces before full hardening.
- The docs UI is currently a lightweight renderer.
- Mermaid diagrams may need a full renderer in the React app.
- Future SDK docs describe direction until a package is published.

## Rule

Do not hide limitations. Label them clearly so users trust the platform.
