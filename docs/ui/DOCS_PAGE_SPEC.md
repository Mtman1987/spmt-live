# Docs Page Spec

## Purpose

The Docs page should make SpaceMountain understandable without leaving the app.

## Requirements

- Sidebar navigation from `docs/docs-nav.json`
- Markdown rendering
- Search/filter
- Mermaid diagram support
- Code block styling
- Copy code button
- Breadcrumbs
- Mobile-friendly layout

## Nice To Have

- Version selector
- Last updated metadata
- Feedback button
- Related docs links
- Live API examples
